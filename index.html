Instructions:

Welcome to LockPickle! Your goal is to guess the correct 4-digit code.

A green box means that the number is correct and in the correct position.

A yellow box means that the number is correct but in the wrong position.

A red box means that the number is incorrect.

A blue box means that there are 2 or more of that number in the code but in the wrong positions.

A blue/green gradient box means that there are 2 or more of that number in the code and one of them is already in the correct position.

You can make up to 5 attempts. The color-coded hints will help you find the code. Good luck!
